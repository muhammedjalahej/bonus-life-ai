To make Login, Register, and Forgot password work:

1. Start the dev mail server (so forgot-password "emails" are received)
   - Double-click: run_dev_mail.bat
   Keep this window open. Reset links will appear here and in mail_output folder.

2. Start the backend
   - Double-click: run_backend.bat
   (Runs on port 8001; frontend uses 8001.)

3. Start the frontend
   From app/frontend/insulyn-frontend run: npm run dev
   Open http://localhost:5173

4. Forgot password: use it in the app; the reset link will show in the
   run_dev_mail window and in app/backend/mail_output/*.eml. Open the link to set a new password.

5. Admin panel: run once: python make_admin.py your@email.com
   Then log in with that email and choose the "Admin" toggle.

6. Real email (Gmail/Hotmail): see EMAIL_SETUP.md. Set SMTP_* in .env and restart backend.
